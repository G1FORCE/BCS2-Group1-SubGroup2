**Social Network — Web System**

A full-featured social networking web system developed collaboratively by a team of Computer Science students. 
The platform allows users to create profiles, connect with friends, share posts, and interact through likes and comments in real time. 
Built as an academic group project, it demonstrates practical implementation of core web development concepts including user authentication, 
database management, RESTful API design, and responsive front-end interfaces. The project was managed using Git for version control and GitHub for collaboration, 
with each team member contributing across different modules of the system. This was a hands-on exercise in software engineering teamwork, agile-style task division, 
and building a scalable multi-user web application from the ground up.
